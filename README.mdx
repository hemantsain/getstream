# Get Random Users

This project will get random users data and display in a decent list. You can get more detail info of the users via long press
on any user or simply via clicking on any user.
The user list contains user image, name and phone number.
The detail view of user contains user name, email, city, DOB and gender.
To check out this project, lets start by cloning the repository:

```
git clone https://github.com/hemantsain/getstream.git
cd getstream
yarn
npx pod-install
```

## How to run the App

- iOS
  - run command 
  ```
  yarn ios 
  or
  npx react-native run-ios
  ```

- Android
  - run command 
  ```
  yarn android
  or
  npx react-native run-android
  ```
